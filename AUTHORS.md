# Authors and Contributors

This package was developed by:

- Debopriyo Biswas <d.biswas@duke.edu>
- Diana Munoz-Valencia <dimunozv@umd.edu>

The original scripts were developed as part of an internal research workflow for submitting quantum circuits to IonQ machines, retrieving raw data, unpacking results, and assessing data quality in real time.

The circuit submission and data retrieval parts were written by Diana, and the data unpacking and fidelity plotting were written by Debopriyo. Testing and review were done by both Diana and Debopriyo.

This public repository contains a cleaned and reorganized version of those tools for broader use.
